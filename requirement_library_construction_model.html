REQ-IAM-001 · v1.0
"All service and privileged accounts must be provisioned and deprovisioned through Okta. Applications may not maintain local credential stores. Accounts must be reviewed quarterly. Access must be revoked within 24 hours of role change or termination. Verified by: IAM audit report showing zero unregistered accounts; quarterly review sign-off on file."
Grouping rule — when to combine controls
✓
Same implementation team and tool — AC-2 + AC-2(1) both owned by IAM team in Okta → one requirement
✓
Same verification method — both verified by the same audit report → one requirement
✗
Different audiences — AC-2 (app dev) vs SC-28 (infra/Terraform) → separate requirements
✗
Different tools or systems → separate requirements
✗
Different verification cadence or method → separate requirements
